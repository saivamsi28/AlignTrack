 ⚛️ AlignTrack

**Enterprise Goal Alignment & Tracking Platform**

AlignTrack is a modern, role-based SaaS platform designed to streamline corporate goal setting, manager approvals, and quarterly performance tracking. Built as a fast, edge-deployed Single Page Application (SPA).

## ✨ Key Features

* **Role-Based Access Control:** Dedicated, secure portals for Employees, Managers, and HR Admins.
* **Client-Side Math Engine:** Dynamically calculates goal weightages, target tracking (Min, Max, Zero Error), and prevents out-of-bounds submissions.
* **Workflow Governance:** Hard locks for 100% weightage validation, maximum of 8 goals, and strict manager approval pipelines.
* **Phased Lifecycle:** Toggle between Phase 1 (Setup) and Q1-Q4 Check-ins.
* **Live Admin Dashboard:** Real-time organizational completion metrics and one-click CSV report generation.

## 🛠 Tech Stack

* **Frontend:** React (TypeScript), React Router DOM
* **Styling:** Vanilla CSS (Enterprise Indigo Theme)
* **Architecture:** Decoupled SPA with a Client-Side Math Engine to optimize server compute costs.
* **Deployment:** Vercel Edge
